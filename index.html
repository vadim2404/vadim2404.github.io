1. Installation

1.1 OS package

# apt-get install phpunit

1.2 phpunit.phar

$ wget https://phar.phpunit.de/phpunit.phar
$ chmod +x phpunit.phar

1.3 Composer

$ ./composer.phar require 'phpunit/phpunit=4.0.*'

1.4 PEAR

# pear config-set auto_discover 1
# pear install pear.phpunit.de/PHPUnit
